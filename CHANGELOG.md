# Changelog

## 0.1.0 (2026-04-06)

Initial open-source release.

### Features
- SSH terminal with xterm.js, full escape sequence support
- SFTP file browser with tree view and lazy loading
- Code editor with CodeMirror 6, 100+ language syntax highlighting
- Markdown preview with GFM (remark/rehype pipeline)
- SSH key generation (Ed25519, RSA) and import
- Extra key bar with terminal shortcuts (Esc, Tab, Ctrl combos, Snippets)
- Pinch zoom for terminal and editor
- Workspace management (add, edit, connect)
- Dark/light theme support (Catppuccin)
- DI-based plugin system for tabs, panels, settings, and markdown extensions
- Policy API for feature gating (defaults: unlimited)
- Overlay hooks for native view coordination
- Session save/load hooks
